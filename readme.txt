Image ALT Fixer automatically sets the ALT attribute content of all IMG and FIGURE tags, present in a post, to improve SEO and always ensure your authority on search engines.

== Description ==

Image ALT Fixer automatically sets the ALT attribute content of all IMG and FIGURE tags, present in a post, with a keyphrase calculated based on the post content.

== Installation ==

The plugin is simple to install:

1. Download 'image-alt-fixer.zip'
2. Unzip
3. Upload 'image-alt-fixer' directory to your '/wp-content/plugins' directory, or install the plugin throught the Wordpress plugins screen directly.
4. Activate the plugin through the 'Plugins' menu in WordPress.
